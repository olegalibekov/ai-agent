## File Descriptions

### Core Components

**god_agent.py** (800+ lines)
- Main GodAgent class
- Input processing pipeline
- Claude API integration
- Tool orchestration
- Context management
- Text and voice modes

**voice_module.py** (250+ lines)
- VoiceInterface class
- Whisper API for STT
- OpenAI TTS for voice output
- Wake word detection
- Audio recording and playback

**rag_engine.py** (400+ lines)
- RAGEngine class
- FAISS vector store
- Document indexing
- Similarity search
- Persistence layer

**mcp_manager.py** (600+ lines)
- MCPManager class
- Tool registry and execution
- FilesystemTool
- WebSearchTool
- GitHubTool (stub)
- CalendarTool (stub)
- EmailTool (stub)
- SlackTool (stub)

**memory.py** (450+ lines)
- MemoryManager class
- Short-term conversation history
- Long-term SQLite storage
- Fact extraction
- Session tracking
- Statistics

**task_tracker.py** (500+ lines)
- TaskTracker class
- Task CRUD operations
- Priority and status management
- Filtering and search
- Statistics

**cli.py** (350+ lines)
- Typer-based CLI
- Commands: start, task-list, task-create, stats, info
- Rich console output

## Key Features

### 1. Voice Interface
- Whisper for speech-to-text
- OpenAI TTS for voice output
- Wake word detection
- Multi-language support

### 2. RAG Engine
- FAISS vector store
- 768-dim embeddings
- Automatic chunking
- Similarity search

### 3. MCP Tools
- Filesystem operations
- Web search
- GitHub (planned)
- Calendar (planned)
- Email (planned)

### 4. Memory System
- Short-term (in-memory)
- Long-term (SQLite)
- Fact extraction
- Session tracking

### 5. Task Management
- CRUD operations
- Priority levels
- Status tracking
- Tags and filters
- Statistics

### 6. CLI Interface
- Interactive modes
- Task commands
- Statistics
- Rich output

## Dependencies

### AI/ML
- anthropic (Claude API)
- openai (Whisper, TTS)
- langchain (orchestration)
- sentence-transformers (embeddings)
- faiss-cpu (vector search)

### Audio
- pyaudio (recording)
- SpeechRecognition (STT)
- pydub (processing)

### Storage
- sqlite3 (database)
- pickle (serialization)

### Utils
- rich (terminal UI)
- typer (CLI)
- pydantic (validation)
- pyyaml (config)

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Configure .env with API keys
3. Run: `python cli.py start`
